## 为什么会出现安全警告？

### Windows SmartScreen

Windows使用 **SmartScreen** 筛选器保护用户免受未知软件的潜在威胁。警告出现的原因：

1. **程序未签名**
   - 本程序没有数字签名证书（证书费用约$100-400/年）
   - Windows无法验证发布者身份

2. **首次运行**
   - 程序是新发布的，还没有建立"声誉"
   - SmartScreen依赖下载量和运行统计

3. **从互联网下载**
   - Windows会标记从网络下载的可执行文件
   - 触发额外的安全检查

**重要提示**：这些警告**不意味着**程序有病毒或恶意代码！

## 如何安全运行程序？

### 方法一：通过SmartScreen警告运行（推荐）

当看到"Windows已保护你的电脑"警告时：

1. 点击 **"更多信息"**
2. 点击 **"仍要运行"** 按钮
3. 程序将正常启动

![SmartScreen Warning](https://via.placeholder.com/600x300?text=SmartScreen+%E8%AD%A6%E5%91%8A%E7%A4%BA%E4%BE%8B)

### 方法二：解除文件锁定

如果文件被标记为"已阻止"：

1. **右键点击** `ARONA.exe`
2. 选择 **"属性"**
3. 在底部找到 **"安全性"** 部分
4. 勾选 **"解除锁定"** 复选框
5. 点击 **"确定"**
6. 双击运行程序

![Unblock File](https://via.placeholder.com/400x500?text=%E8%A7%A3%E9%99%A4%E9%94%81%E5%AE%9A%E7%A4%BA%E4%BE%8B)

### 方法三：从源码编译（最安全）

如果你担心安全性，可以自己编译：

```bash
# 1. 克隆源代码
git clone https://github.com/QiShao42/BA_StarryPat.git
cd BA_StarryPat

# 2. 配置并编译
mkdir build
cd build
cmake .. -G "Ninja" -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release

# 3. 运行
.\ARONA.exe
```

详见 [README.md](README.md#构建指南)

## 常见警告类型

### 1. Windows Defender SmartScreen

**警告信息**：
```
Windows 已保护你的电脑
Microsoft Defender SmartScreen 已阻止来自未知发布者的应用启动
```

**解决方案**：点击"更多信息" → "仍要运行"

### 2. 用户账户控制 (UAC)

**警告信息**：
```
你要允许此应用对你的设备进行更改吗？
未验证的发布者
```

**解决方案**：点击"是"（程序不需要管理员权限，但可能被UAC拦截）

### 3. 杀毒软件误报

某些杀毒软件可能将程序标记为：
- PUA (Potentially Unwanted Application) - 潜在不需要的应用
- Generic Trojan - 通用木马特征

**原因**：
- 程序使用截图功能
- 程序使用自动化模拟点击
- 程序行为类似游戏外挂工具

**解决方案**：添加到杀毒软件的白名单/排除列表

## 开发者说明

### 为什么不购买代码签名证书？

1. **成本问题**
   - 标准证书：约 $100-300/年
   - EV证书：约 $300-500/年
   - 对于个人开源项目成本较高

2. **流程复杂**
   - 需要公司或个人身份验证
   - 审核周期长
   - 需要实体USB密钥（EV证书）

3. **开源透明性**
   - 完整源代码公开
   - 用户可自行审查和编译
   - 这比签名更值得信任
