# 学生头像模板使用说明

## 概述

本程序支持从外部文件夹加载学生头像模板，这样您就可以在游戏更新后自行添加新学生的头像，而无需等待程序更新。

## 模板文件夹结构

在程序所在目录下创建以下文件夹：
```
Arona.exe所在目录/
└── templates/
    └── student_avatar/
        ├── 学生名称1.png
        ├── 学生名称2.png
        └── ...
```

## 如何添加新模板

### 1. 截取模板的方法

1. 将目标学生精选（不能多于5个），进入游戏的邀请界面
2. 选择调试功能：截图调试，点击开始调试
3. 图片自动保存在同路径screenshots文件夹中

### 2. 准备模板图片

模板图片应该是学生头像的截图，要求：
- **格式**：PNG格式
- **命名**：文件名为学生名称（不含扩展名），例如 `妮露.png`

### 3. 放置模板文件

将准备好的PNG文件放入 `templates/student_avatar/` 文件夹中。

### 4. 重新加载模板

在程序界面中点击 **"重新加载模板"** 按钮，程序会自动加载新添加的模板。

## 注意事项

- 如果识别效果不佳，可以尝试重新截取模板，注意不要滑动邀请界面

## 常见问题

**Q: 添加模板后没有生效？**
A: 请点击"重新加载模板"按钮，或重启程序。

**Q: 模板识别不准确？**
A: 联系开发者。

**Q: 如何知道模板加载成功？**
A: 查看程序日志，会显示"学生头像模板加载完成，共加载X个模板（外部X个，内置X个）"。

**Q: 可以删除内置模板吗？**
A: 不可以删除内置模板，外部模板会覆盖同名的内置模板。

## 示例

假设游戏新增了学生"阿罗娜（泳装）"：

1. 在 `templates/student_avatar/` 文件夹中放入 `阿罗娜（泳装）.png`
2. 点击程序中的"重新加载模板"按钮
3. 在"邀请学生设置"中即可看到新增的学生
4. 配置完成后即可使用邀请功能
